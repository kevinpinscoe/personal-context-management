# obsidian/writing-prompts/ — Writing Prompts and Creative Seeds

Writing prompts, creative starting points, and generative seeds for future writing
work — blog posts, essays, documentation drafts, or personal journaling.

## What Belongs Here

- Prompts generated by AI during ingest sessions worth returning to
- Creative angles on technical topics that would make good posts or essays
- Documentation topics flagged during reading that need to be written
- Personal reflection prompts tied to active interests

## Frontmatter Template

```yaml
---
title: ""
created: YYYY-MM-DD
category: writing-prompts
type: ""          # blog-post | essay | documentation | journal | other
status: ""        # open | drafted | published | abandoned
sources:
  - https://...
tags:
  - pcm-output
  - writing-prompt
---
```

## Content Notes

- A prompt here is an open invitation — not a commitment.
- Mark `status: abandoned` rather than deleting when a prompt stops being relevant.
  Abandoned prompts are useful context for what was considered but set aside.
- If a prompt gets written, update `status: published` and add the published URL
  to the frontmatter.
