# obsidian/ai/ — AI Tools, Models, Agents, and Workflows

Distilled knowledge about artificial intelligence tools, large language models,
agent frameworks, prompt engineering, and AI-assisted workflows.

## What Belongs Here

- Summaries of new models, capabilities, or providers worth tracking
- Patterns or techniques for building agents and multi-step AI workflows
- Prompt engineering practices that produced good results
- Tool evaluations: MCP servers, AI coding assistants, integrations
- Workflow changes enabled by AI (not just theory — actionable changes)

## Content Notes

- Prefer actionable conclusions over article summaries.
- If a model or tool is worth evaluating, say so explicitly and note what to test.
- Sub-organize with subdirectories by topic (e.g. `ai/agents/`, `ai/models/`) once
  enough files accumulate in a single area.
