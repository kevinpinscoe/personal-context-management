# obsidian/mindmaps/ — Visual Thinking and Concept Maps

Structured concept maps, relationship diagrams, and visual thinking outputs.

## What Belongs Here

- Mermaid diagrams, ASCII maps, or linked concept outlines
- Relationship maps between ideas surfaced during ingest
- Mental models worth preserving as structured visual references

## Frontmatter Template

```yaml
---
title: ""
created: YYYY-MM-DD
category: mindmaps
sources:
  - https://...
tags:
  - pcm-output
  - mindmap
---
```

## Content Notes

- Prefer Mermaid diagrams (render natively in Obsidian with the Mermaid plugin).
- If a mindmap came from an AI session, note the prompt that generated it in
  the `sources` field so it can be reproduced or extended.
- Mindmaps that belong clearly to another domain (`ai/`, `infrastructure/`, etc.)
  can live there as embedded diagrams instead — this directory is for maps that
  stand alone as the primary artifact.
